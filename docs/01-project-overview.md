# 待办事项管理系统 - 项目概述

> **最後更新**: 2026-02-02 | **版本**: v1.0.0

## 📌 项目简介

一个功能完整的全栈待办事项管理网站，参照专案管理模式，支持任务状态流转、任务依赖关系、看板列顺序锁定等功能。

## 🎯 核心功能

- 用户注册与登录
- 多项目管理
- 看板视图（Kanban Board）
- 任务状态流转（Backlog → Ready → In Progress → Review → Done）
- 任务依赖系统
- 看板列顺序锁定
- 任务优先级
- 截止日期
- 成员分配
- 深色/浅色主题（支持系统跟随）
- RWD 响应式设计
- 个人设置页面（头像上传）

## 👥 目标用户

- 个人用户
- 小型团队（最多 1000 用户规模）
